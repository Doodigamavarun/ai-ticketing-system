# AI Ticketing System

## Overview
Smart AI-based ticketing system that auto resolves or assigns tickets.

## Tech Stack
- FastAPI
- React
- SQLite

## Setup

### Backend
```
cd backend
uvicorn main:app --reload
```

### Frontend
```
cd frontend
npm install
npm start
```

## Features
- AI Analysis
- Auto Resolution
- Ticket Routing
- Employee Directory
- Analytics Dashboard
